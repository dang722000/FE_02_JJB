  • select 코드 - option을 추가해서 사용

    • value 코드로 태그해서 나중에 이 부분만을 수정할떄 용이하게

    selected 코드로 기본적으로 어떤 값을 선택할지

    multiple 코드를 추가하면 여러게가 보이고 size코드를 더 추가하면 몇개 가보일지 설정가능

  • textarea 코드를 사용하면 주관식 택스트박스가 들어간다

    • cols 코드를 추가해 가로로 글자수 제한을 설정하고

    rows 코드를 추가해서 세로로 글자수 제한을 설정함

    하지만 textarea 코드는 cols나 rows를 추가하지않고도 사용가능하면 사용자로 하여금 크기도 조절이 가능함
